## 2. G-变换方法

定理8: ${A}_{n \times  n}$ 可逆 $\Rightarrow  \exists$ 有限个G-矩阵之积T,使得TA 为可逆上三角矩阵。

证明: 以n=4为例

$$
\left| A\right|  \neq  0\;{\mathbf{\beta }}^{\left( 0\right) } = \left\lbrack  \begin{array}{l} {a}_{11} \\  {a}_{21} \\  {a}_{31} \\  {a}_{41} \end{array}\right\rbrack   \neq  0
$$

$\Rightarrow  \exists$ 有限个 $\mathrm{G}$ -矩阵之积 ${\mathrm{T}}_{0}$ ,使得

$$
{T}_{0}{\mathbf{\beta }}^{\left( 0\right) } = \left\lbrack  \begin{matrix} \left| {\mathbf{\beta }}^{\left( 0\right) }\right| \\  0 \\  0 \\  0 \end{matrix}\right\rbrack  ,{a}_{11}^{\left( 1\right) } = \left| {\mathbf{\beta }}^{\left( 0\right) }\right|  > 0
$$

${T}_{0}A = \left\lbrack  \begin{matrix} {a}_{11}^{\left( 1\right) } & {a}_{12}^{\left( 1\right) } & {a}_{13}^{\left( 1\right) } & {a}_{14}^{\left( 1\right) } \\  0 & {a}_{22}^{\left( 1\right) } & {a}_{23}^{\left( 1\right) } & {a}_{24}^{\left( 1\right) } \\  0 & {a}_{32}^{\left( 1\right) } & {a}_{33}^{\left( 1\right) } & {a}_{34}^{\left( 1\right) } \\  0 & {a}_{42}^{\left( 1\right) } & {a}_{43}^{\left( 1\right) } & {a}_{44}^{\left( 1\right) } \end{matrix}\right\rbrack$

$$
\left\lbrack  \begin{matrix} {a}_{22}^{\left( 1\right) } & {a}_{23}^{\left( 1\right) } & {a}_{24}^{\left( 1\right) } \\  {a}_{32}^{\left( 1\right) } & {a}_{33}^{\left( 1\right) } & {a}_{34}^{\left( 1\right) } \\  {a}_{42}^{\left( 1\right) } & {a}_{43}^{\left( 1\right) } & {a}_{44}^{\left( 1\right) } \end{matrix}\right\rbrack   = {A}^{\left( 1\right) }
$$

$\left| {\mathrm{A}}^{\left( 1\right) }\right|  \neq  0,\;{\mathbf{\beta }}^{\left( 1\right) } = \left\lbrack  \begin{array}{l} {\mathrm{a}}_{22}^{\left( 1\right) } \\  {\mathrm{a}}_{32}^{\left( 1\right) } \\  {\mathrm{a}}_{42}^{\left( 1\right) } \end{array}\right\rbrack   \neq  0\; \Rightarrow  \exists$ 有限个G-矩阵之积 ${\mathrm{T}}_{1}$ ,使得

$$
{\mathrm{T}}_{1}{\mathbf{\beta }}^{\left( 1\right) } = \left\lbrack  \begin{matrix} \left| {\mathbf{\beta }}^{\left( 1\right) }\right| \\  0 \\  0 \end{matrix}\right\rbrack  ,{\mathrm{a}}_{22}^{\left( 2\right) } = \left| {\mathbf{\beta }}^{\left( 1\right) }\right|  > 0
$$

$$

$$

$\left| {\mathrm{A}}^{\left( 2\right) }\right|  \neq  0,{\mathbf{\beta }}^{\left( 2\right) } = \left\lbrack  \begin{array}{l} {\mathrm{a}}_{33}^{\left( 2\right) } \\  {\mathrm{a}}_{43}^{\left( 2\right) } \end{array}\right\rbrack   \neq  0 \Rightarrow  \exists \mathrm{G}$ -矩阵 ${\mathrm{T}}_{2}$ ,使得

$$
{T}_{2}{\mathbf{\beta }}^{\left( 2\right) } = \left\lbrack  \begin{matrix} \left| {\mathbf{\beta }}^{\left( 2\right) }\right| \\  0 \end{matrix}\right\rbrack  ,{a}_{33}^{\left( 3\right) } = \left| {\mathbf{\beta }}^{\left( 2\right) }\right|  > 0\;{T}_{2}{A}^{\left( 2\right) } = \left\lbrack  \begin{matrix} {a}_{33}^{\left( 3\right) } & {a}_{34}^{\left( 3\right) } \\  0 & {a}_{44}^{\left( 3\right) } \end{matrix}\right\rbrack
$$

$$
T = \left\lbrack  \begin{matrix} {I}_{2} & \\   & {T}_{2} \end{matrix}\right\rbrack  \left\lbrack  \begin{matrix} 1 & \\   & {T}_{1} \end{matrix}\right\rbrack  {T}_{0}\;\text{ TA } = \left\lbrack  \begin{matrix} {a}_{11}^{\left( 1\right) } & {a}_{12}^{\left( 1\right) } & {a}_{13}^{\left( 1\right) } & {a}_{14}^{\left( 1\right) } \\   & {a}_{22}^{\left( 2\right) } & {a}_{23}^{\left( 2\right) } & {a}_{24}^{\left( 2\right) } \\   & & {a}_{33}^{\left( 3\right) } & {a}_{34}^{\left( 3\right) } \\   & & & {a}_{44}^{\left( 3\right) } \end{matrix}\right\rbrack   \triangleq  R

## 2. G-变换方法

定理8: ${A}_{n \times  n}$ 可逆 $\Rightarrow  \exists$ 有限个G-矩阵之积T,使得TA 为可逆上三角矩阵。

[注]: $\det \mathrm{T} = 1 \Rightarrow  \det \mathrm{A} = {\mathrm{a}}_{11}^{\left( 1\right) }{\mathrm{a}}_{22}^{\left( 2\right) }\cdots {\mathrm{a}}_{\mathrm{n} - 1,\mathrm{n} - 1}^{\left( \mathrm{n} - 1\right) }{\mathrm{a}}_{\mathrm{n},\mathrm{n}}^{\left( \mathrm{n} - 1\right) }$

因此 ${\mathrm{a}}_{\mathrm{n},\mathrm{n}}^{\left( \mathrm{n} - 1\right) }$ 与 $\det \mathrm{A}$ 同符号

当 ${\mathrm{A}}_{\mathrm{n},\mathrm{n}}$ 不可逆时,仍可得 $\mathrm{{TA}} = \mathrm{R}$ ,但 $\mathrm{R}$ 是不可逆矩阵

$$

则 $Q = {T}^{-1} = {T}^{\mathrm{T}}, R = \left\lbrack  \begin{array}{rrr} 5 & 3 & 7 \\  5 & {16}/5 & 8 \\   & {13}/5 &  \end{array}\right\rbrack   : A = {QR}$
