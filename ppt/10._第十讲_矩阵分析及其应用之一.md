## 敛散性

定义: 将矩阵序列 ${A}^{\left( k\right) } = {\left\lbrack  {a}_{ij}^{\left( k\right) }\right\rbrack  }_{m \times  n}$ ,记作 $\left\{  {A}^{\left( k\right) }\right\}$ ,当

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{a}}_{\mathrm{{ij}}}^{\left( \mathrm{k}\right) } = {\mathrm{a}}_{\mathrm{{ij}}}\left( {\forall \mathrm{i},\mathrm{j}}\right)
$$

时,称矩阵序列 $\left\{  {A}^{\left( k\right) }\right\}$ 收敛于矩阵 $A = \left( {a}_{ij}\right)$ 。记作

$\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = \mathrm{A}$ ,或者 ${\mathrm{A}}^{\left( \mathrm{k}\right) } \rightarrow  \mathrm{A}\left( {\mathrm{k} \rightarrow  \infty }\right)$

若数列 $\left( {a}_{ij}^{\left( k\right) }\right)$ 之一发散,称 $\left\{  {A}^{\left( k\right) }\right\}$ 发散

性质:

(1)若 $\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = {\mathrm{A}}_{\mathrm{m} \times  \mathrm{n}},\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{B}}^{\left( \mathrm{k}\right) } = {\mathrm{B}}_{\mathrm{m} \times  \mathrm{n}}$ 则

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\left( {{\mathrm{{aA}}}^{\left( \mathrm{k}\right) } + {\mathrm{{bB}}}^{\left( \mathrm{k}\right) }}\right)  = \mathrm{{aA}} + \mathrm{{bB}},\;\forall \mathrm{a},\mathrm{b}
$$

(2) 若 $\mathop{\lim }\limits_{{k \rightarrow  \infty }}{A}^{\left( k\right) } = {A}_{m \times  n},\mathop{\lim }\limits_{{k \rightarrow  \infty }}{B}^{\left( k\right) } = {B}_{n \times  l}$ 则

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\left( {{\mathrm{A}}^{\left( \mathrm{k}\right) }{\mathrm{B}}^{\left( \mathrm{k}\right) }}\right)  = \mathrm{{AB}}
$$

(3) 若 ${A}^{\left( k\right) }$ 与 $A$ 是可逆矩阵,且 $\mathop{\lim }\limits_{{k \rightarrow  \infty }}{A}^{\left( k\right) } = A$ ,则

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\left( {\mathrm{A}}^{\left( \mathrm{k}\right) }\right) }^{-1} = {\mathrm{A}}^{-1}
$$

定理1: 设 ${A}^{\left( k\right) }, A \in  {C}^{\text{ mxn }}$ ，则

$$
\forall \parallel  \bullet  \parallel ,\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\begin{Vmatrix}{\mathrm{A}}^{\left( \mathrm{k}\right) }\end{Vmatrix} = 0
$$

$$
\forall \parallel  \bullet  \parallel ,\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\begin{Vmatrix}{{\mathrm{A}}^{\left( \mathrm{k}\right) } - \mathrm{A}}\end{Vmatrix} = 0
$$

(1) $\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = 0$

(2) $\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = \mathrm{A}$

证明: (1)考虑F -矩阵范数

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = 0
$$

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{a}}_{\mathrm{{ij}}}^{\left( \mathrm{k}\right) } = 0\;\left( {\text{ all }\mathrm{i},\mathrm{j}}\right)
$$

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\mathop{\sum }\limits_{{\mathrm{i} = 1}}^{\mathrm{m}}\mathop{\sum }\limits_{{\mathrm{j} = 1}}^{\mathrm{n}}{\left| {\mathrm{a}}_{\mathrm{{ij}}}^{\left( \mathrm{k}\right) }\right| }^{2} = 0
$$

$$
\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\begin{Vmatrix}{\mathrm{A}}^{\left( \mathrm{k}\right) }\end{Vmatrix}}_{\mathrm{F}} = 0
$$

(2)由 $\mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}{\mathrm{A}}^{\left( \mathrm{k}\right) } = \mathrm{A} \Leftrightarrow  \mathop{\lim }\limits_{{\mathrm{k} \rightarrow  \infty }}\left( {{\mathrm{A}}^{\left( \mathrm{k}\right) } - \mathrm{A}}\right)  = 0\;$ 可直接的得到

敛散性的另一定义: 矩阵序列 $\left\{  {A}^{\left( k\right) }\right\}$ 收敛的充要条件为对任给 $\varepsilon  > 0$ 存在 $\mathrm{N}\left( \varepsilon \right)$ ,当 $\mathrm{k},1 \geq  \mathrm{N}\left( \varepsilon \right)$ 时有

$$
\begin{Vmatrix}{{\mathrm{A}}^{\left( \mathrm{k}\right) } - {\mathrm{A}}^{\left( 1\right) }}\end{Vmatrix} < \varepsilon
$$

其中 $\parallel  \bullet  \parallel$ 为任意的广义矩阵范数。

例1: ${A}^{\left( n\right) } = \left( \begin{matrix} 1 - \frac{1}{n} & \sin \left( \frac{1}{n}\right) \\  {e}^{-n} & \frac{n}{k}\frac{\sin \left( k\right) }{{k}^{2}} \end{matrix}\right)$ ,证明其敛散性

因为求不出 ${A}^{\left( n\right) }$ 的极限从而很难应用定义证明收敛。

相反,由于 $\left| {\mathop{\sum }\limits_{{k = m + 1}}^{n}\frac{\sin \left( k\right) }{{k}^{2}}}\right|  < \left| {\mathop{\sum }\limits_{{k = m + 1}}^{n}\frac{1}{{k}^{2}}}\right|  < \left| {\mathop{\sum }\limits_{{k = m + 1}}^{n}\frac{1}{k\left( {k - 1}\right) }}\right|  < \frac{1}{m}$

从而只要取1充分大,则当 $m, n > 1$ 时就有

$$
\left| {\mathop{\sum }\limits_{{\mathrm{k} = \mathrm{m} + 1}}^{\mathrm{n}}\frac{\sin \left( \mathrm{k}\right) }{{\mathrm{k}}^{2}}}\right|  \leq  \varepsilon
$$

$\because \frac{1}{\mathrm{k}\left( {\mathrm{k} - 1}\right) } = \frac{1}{\mathrm{k} - 1} - \frac{1}{\mathrm{k}}$

这样 ${A}^{\left( n\right) }$ 收敛

$\mathop{\sum }\limits_{{k = m + 1}}^{n}\frac{1}{k\left( {k - 1}\right) } = \frac{1}{m} - \frac{1}{n} < \frac{1}{m}$

定义: 若 ${A}_{n \times  n}$ 满足 $\mathop{\lim }\limits_{{k \rightarrow  \infty }}{A}^{k} = {0}_{n \times  n}$ ,称 $A$ 为收敛矩阵

定理2: $\mathrm{A}$ 为收敛矩阵 $\rho \left( \mathrm{A}\right)  < 1$

证明: 充分性: 已知 $\rho \left( \mathrm{A}\right)  < 1$ ,对 $\varepsilon  = \frac{1}{2}\left\lbrack  {1 - \rho \left( \mathrm{A}\right) }\right\rbrack   > 0$

存在矩阵范数 $\parallel  \bullet  {\parallel }_{\mathrm{M}}$ ,使得

$$
\parallel \mathrm{A}{\parallel }_{\mathrm{M}} \leq  \rho \left( \mathrm{A}\right)  + \varepsilon  = \frac{1}{2}\left\lbrack  {1 + \rho \left( \mathrm{A}\right) }\right\rbrack   < 1
$$

于是有 ${\begin{Vmatrix}{A}^{k}\end{Vmatrix}}_{M} \leq  \parallel A{\parallel }_{M}^{k} \rightarrow  0$ ,故由定理 1 可得 ${A}^{k} \rightarrow  0$

必要性: 已知 ${A}^{k} \rightarrow  0$ ,设 ${Ax} = {\lambda x}\left( {x \neq  0}\right)$ ,则有

$$
{\lambda }^{\mathrm{k}}\mathrm{x} = {\mathrm{A}}^{\mathrm{k}}\mathrm{x} \rightarrow  0\; \Rightarrow  {\lambda }^{\mathrm{k}} \rightarrow  0\; \Rightarrow  \left| \lambda \right|  < 1
$$

故 $\rho \left( \mathrm{A}\right)  < 1$

定理3: 若矩阵范数 $\parallel  \cdot  {\parallel }_{\mathrm{M}}$ 使 $\parallel \mathrm{A}{\parallel }_{\mathrm{M}} < 1$ ,则 ${\mathrm{\;A}}^{\mathrm{k}} \rightarrow  0$

证明: $\;\rho \left( \mathrm{A}\right)  \leq  \parallel \mathrm{A}{\parallel }_{\mathrm{M}} < 1 \Rightarrow  {\mathrm{A}}^{\mathrm{k}} \rightarrow  0$

例: $A = \left\lbrack  \begin{array}{ll} {0.1} & {0.3} \\  {0.7} & {0.6} \end{array}\right\rbrack$

$$
\parallel \mathrm{A}{\parallel }_{1} = {0.9} < 1 \Rightarrow  {\mathrm{A}}^{\mathrm{k}} \rightarrow  0
$$

定义:设有矩阵序列 $\left\{  {A}^{\left( k\right) }\right\}$ ，其中 ${A}^{\left( k\right) } = \left\lbrack  {a}_{ij}^{\left( k\right) }\right\rbrack   \in  {C}^{m \times  n}$

称 ${A}^{\left( 0\right) } + {A}^{\left( 1\right) } + \cdots  + {A}^{\left( k\right) } + \cdots$ 为矩阵级数。记为 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{A}^{\left( k\right) }$

部分和 ${S}^{\left( N\right) } = \mathop{\sum }\limits_{{k = 0}}^{N}{A}^{\left( k\right) }$ 构成矩阵序列 $\left\{  {S}^{\left( N\right) }\right\}$

敛散性: 若 $\mathop{\lim }\limits_{{N \rightarrow  \infty }}{S}^{\left( N\right) } = S$ ,称 $\sum {A}^{\left( k\right) }$ 收敛于 $S$ ,记做

$$
\sum {\mathrm{A}}^{\left( \mathrm{k}\right) } = \mathrm{S}
$$

若 $\left\{  {S}^{\left( N\right) }\right\}$ 发散,称 $\sum {A}^{\left( k\right) }$ 发散

性质 1: $\sum {\mathrm{A}}^{\left( \mathrm{k}\right) } = \mathrm{S}$

证明: 左 $\mathop{\lim }\limits_{{\mathrm{N} \rightarrow  \infty }}{\mathrm{S}}^{\left( \mathrm{N}\right) } = \mathrm{S}$

$\mathop{\lim }\limits_{{\mathrm{N} \rightarrow  \infty }}{\mathrm{S}}_{\mathrm{{ij}}}^{\left( \mathrm{N}\right) } = {\mathrm{S}}_{\mathrm{{ij}}}\left( {\mathrm{{all}}\;\mathrm{i},\mathrm{j}}\right)$

性质 2: 若 $\sum \left| {a}_{ij}^{\left( k\right) }\right|$ 收敛 (all i, j),称 $\sum {A}^{\left( k\right) }$ 绝对收敛。

(1) $\sum {A}^{\left( k\right) }$ 绝对收敛 $\Rightarrow  \sum {A}^{\left( k\right) }$ 收敛

(2)若 $\sum {A}^{\left( k\right) }$ 绝对收敛于 $S$ ,对 $\sum {A}^{\left( k\right) }$ 任意重组

重排得 $\sum {B}^{\left( k\right) }$ ,则 $\sum {B}^{\left( k\right) }$ 绝对收敛于 $S$ 。

性质3: $\sum {A}^{\left( k\right) } = S$ 绝对收敛 $\Leftrightarrow  \forall \parallel  \bullet  \parallel ,\sum \begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}$ 收敛

证明: 只需考虑矩阵范数 $\parallel  \bullet  {\parallel }_{m1}$

证明: (必要性) 由于矩阵级数 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{A}^{\left( k\right) }$ 绝对收敛,因此mn个数项级数 ___ $\mathop{\sum }\limits_{{k = 0}}^{\infty }{a}_{ij}^{\left( k\right) }$ 绝对收敛。存在 $\widetilde{M} > 0$ ，使得对任意 $N$ ，都有

$$
\mathop{\sum }\limits_{{\mathrm{k} = 0}}^{\mathrm{N}}\left| {\mathrm{a}}_{\mathrm{{ij}}}^{\left( \mathrm{k}\right) }\right|  < \mathrm{M},\left( {\forall \mathrm{i},\mathrm{j}}\right)
$$

因此 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}}_{m1}$ 收敛

(充分性) $\mathop{\sum }\limits_{{k = 0}}^{\infty }{\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}}_{m1}$ 收敛。因此 $\mathop{\sum }\limits_{{k = 0}}^{\infty }\left| {a}_{ij}^{\left( k\right) }\right|  \leq  \mathop{\sum }\limits_{{k = 0}}^{\infty }{\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}}_{m1}$

故矩阵级数 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{A}^{\left( k\right) }$ 绝对收敛

性质 4: $\sum {A}^{\left( k\right) }$ 收敛于 $S \Rightarrow  \mathop{\sum }\limits_{{k = 0}}^{\infty }P{A}^{\left( k\right) }Q$ 收敛于 PSQ

$\sum {A}^{\left( k\right) }$ 绝对收敛 $\Rightarrow  \mathop{\sum }\limits_{{k = 0}}^{\infty }P{A}^{\left( k\right) }Q$ 绝对收敛

证明: 只需考虑矩阵范数 $\parallel  \cdot  {\parallel }_{m1}$

证明: $\left( 1\right) {S}^{\left( N\right) } = \mathop{\sum }\limits_{{k = 0}}^{N}{A}^{\left( k\right) } \rightarrow  S \Rightarrow  \mathop{\sum }\limits_{{k = 0}}^{N}P{A}^{\left( k\right) }Q = P{S}^{\left( N\right) }Q \rightarrow  {PSQ}$

$$
\Rightarrow  \mathop{\sum }\limits_{{\mathrm{k} = 0}}^{\infty }{\mathrm{{PA}}}^{\left( \mathrm{k}\right) }\mathrm{Q} \rightarrow  \mathrm{{PSQ}}
$$

性质4:

$\sum {A}^{\left( k\right) }$ 绝对收敛 $\Rightarrow  \mathop{\sum }\limits_{{k = 0}}^{\infty }P{A}^{\left( k\right) }Q$ 绝对收敛

(2)矩阵范数 $\parallel  \cdot  \parallel$ ，由性质3知 $\sum \begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}$ 收敛因为 $\begin{Vmatrix}{P{A}^{\left( k\right) }Q}\end{Vmatrix} \leq  \parallel P\parallel \begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}Q\parallel  = M\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}\;\left( {M = \parallel P\parallel \parallel Q\parallel }\right)$ 所以 $\mathop{\sum }\limits_{{k = 0}}^{N}\begin{Vmatrix}{P{A}^{\left( k\right) }Q}\end{Vmatrix} \leq  \mathop{\sum }\limits_{{k = 0}}^{N}\left( {M\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}}\right)  = M\mathop{\sum }\limits_{{k = 0}}^{N}\begin{Vmatrix}{A}^{\left( k\right) }\end{Vmatrix}$ 有界故 $\;\sum {A}^{\left( k\right) }$ 绝对收敛 $\Rightarrow  \mathop{\sum }\limits_{{k = 0}}^{\infty }P{A}^{\left( k\right) }Q$ 绝对收敛

性质 5: $\mathop{\sum }\limits_{{\mathrm{k} = 1}}^{\infty }{\mathrm{A}}^{\left( \mathrm{k}\right) }$ 绝对收敛于 ${\mathrm{S}}_{\mathrm{m} \times  \mathrm{n}},\mathop{\sum }\limits_{{\mathrm{k} = 1}}^{\infty }{\mathrm{B}}^{\left( \mathrm{k}\right) }$ 绝对收敛于 ${\mathrm{T}}_{\mathrm{n} \times  \mathrm{l}}$

则Cauchy积

$$
{A}^{\left( 1\right) }{B}^{\left( 1\right) } + \left\lbrack  {{A}^{\left( 1\right) }{B}^{\left( 2\right) } + {A}^{\left( 2\right) }{B}^{\left( 1\right) }}\right\rbrack   + \left\lbrack  {{A}^{\left( 1\right) }{B}^{\left( 3\right) } + {A}^{\left( 2\right) }{B}^{\left( 2\right) } + {A}^{\left( 3\right) }{B}^{\left( 1\right) }}\right\rbrack
$$

$$
+ \cdots  + \left\lbrack  {{A}^{\left( 1\right) }{B}^{\left( k\right) } + {A}^{\left( 2\right) }{B}^{\left( k - 1\right) } + \cdots  + {A}^{\left( k\right) }{B}^{\left( 1\right) }}\right\rbrack   + \cdots
$$

绝对收敛于ST,记作 $\sum {\mathrm{A}}^{\left( \mathrm{k}\right) } \cdot  \sum {\mathrm{B}}^{\left( \mathrm{k}\right) } = \mathrm{{ST}}$

证明: 只需利用性质3即可

Neumann 级数: ${A}_{n \times  n},\mathop{\sum }\limits_{{k = 0}}^{\infty }{A}^{k},\left( {{A}^{0} = I}\right)$

定理 $4 : {A}_{n \times  n},\sum {A}^{k}$ 收敛 $\Leftrightarrow  {A}^{k} \rightarrow  0$

$\sum {A}^{k}$ 收敛时，其和为 ${\left( I - A\right) }^{-1}$

证明: 必要性。 $\sum {A}^{k}$ 收敛时 $\sum {\left( {A}^{k}\right) }_{ij},\forall i, j$ 收敛

即 ${\left( {A}^{k}\right) }_{ij} \rightarrow  0$ ,也就是 ${A}^{k} \rightarrow  0$

充分性。 ${\mathrm{A}}^{\mathrm{k}} \rightarrow  0$ 由定理 2 可知 $\rho \left( \mathrm{A}\right)  < 1 \Leftrightarrow  \left( {\mathrm{I} - \mathrm{A}}\right)$ 可逆

$$
\left( {\mathrm{I} + \mathrm{A} + {\mathrm{A}}^{2} + \cdots  + {\mathrm{A}}^{\mathrm{N}}}\right) \left( {\mathrm{I} - \mathrm{A}}\right)  = \mathrm{I} - {\mathrm{A}}^{\mathrm{N} + 1}
$$

$$
\left( {\mathrm{I} + \mathrm{A} + {\mathrm{A}}^{2} + \cdots  + {\mathrm{A}}^{\mathrm{N}}}\right)  = {\left( \mathrm{I} - \mathrm{A}\right) }^{-1} - {\mathrm{A}}^{\mathrm{N} + 1}{\left( \mathrm{I} - \mathrm{A}\right) }^{-1} \rightarrow  {\left( \mathrm{I} - \mathrm{A}\right) }^{-1},\mathrm{\;N} \rightarrow  \infty
$$

即 $\sum {A}^{k} = {\left( I - A\right) }^{-1}$

定理 5: ${A}_{n \times  n},\parallel A\parallel  < 1 \Rightarrow  \begin{Vmatrix}{{\left( I - A\right) }^{-1} - \mathop{\sum }\limits_{{k = 0}}^{N}{A}^{k}}\end{Vmatrix} \leq  \frac{\parallel A{\parallel }^{N + 1}}{1 - \parallel A\parallel }, N = 0,1,2$

证明: $\left| A\right|  < 1 \Rightarrow  \rho \left( A\right)  < 1 \Rightarrow  \left( {I - A}\right)$ 可逆

$$
\left( {\mathrm{I} + \mathrm{A} + {\mathrm{A}}^{2} + \cdots  + {\mathrm{A}}^{\mathrm{N}}}\right) \left( {\mathrm{I} - \mathrm{A}}\right)  = \mathrm{I} - {\mathrm{A}}^{\mathrm{N} + 1}
$$

右乘 ${\left( \mathrm{I} - \mathrm{A}\right) }^{-1}$ ,移项可得

$$
{\left( \mathrm{I} - \mathrm{A}\right) }^{-1} - \left( {\mathrm{I} + \mathrm{A} + {\mathrm{A}}^{2} + \cdots  + {\mathrm{A}}^{\mathrm{N}}}\right)  = {\mathrm{A}}^{\mathrm{N} + 1}{\left( \mathrm{I} - \mathrm{A}\right) }^{-1}
$$

恒等式 ${A}^{N + 1} = {A}^{N + 1}{\left( I - A\right) }^{-1}\left( {I - A}\right)  = {A}^{N + 1}{\left( I - A\right) }^{-1} - {A}^{N + 1}{\left( I - A\right) }^{-1}A$

$$
{A}^{N + 1}{\left( I - A\right) }^{-1} = {A}^{N + 1} + {A}^{N + 1}{\left( I - A\right) }^{-1}A
$$

$$
\begin{Vmatrix}{{A}^{N + 1}{\left( I - A\right) }^{-1}}\end{Vmatrix} \leq  \begin{Vmatrix}{A}^{N + 1}\end{Vmatrix} + \begin{Vmatrix}{{A}^{N + 1}{\left( I - A\right) }^{-1}}\end{Vmatrix} \bullet  \begin{Vmatrix}A\end{Vmatrix}
$$

故 $\begin{Vmatrix}{{A}^{N + 1}{\left( I - A\right) }^{-1}}\end{Vmatrix} \leq  \frac{\begin{Vmatrix}{A}^{N + 1}\end{Vmatrix}}{1 - \begin{Vmatrix}A\end{Vmatrix}}\; \Rightarrow  \begin{Vmatrix}{{\left( I - A\right) }^{-1} - \mathop{\sum }\limits_{{k = 0}}^{N}{A}^{k}}\end{Vmatrix} \leq  \frac{\parallel A{\parallel }^{N + 1}}{1 - \parallel A\parallel }$

幂级数:对函数 $f\left( z\right)  = \sum {c}_{k}{z}^{k},\left( {\left| z\right|  < r}\right)$ 方阵 ${A}_{n \times  n}$ ，

构造矩阵幂级数 $f\left( A\right)  = \sum {c}_{k}{A}^{k}$

定理 6: (1) $\rho \left( A\right)  < r \Rightarrow  \sum {c}_{k}{A}^{k}$ 绝对收敛

(2) $\rho \left( A\right)  > r\; \Rightarrow  \sum {c}_{k}{A}^{k}$ 发散

证明: (1) 对 $A$ ,取 $\varepsilon  = \frac{1}{2}\left\lbrack  {r - \rho \left( A\right) }\right\rbrack   > 0,\exists \parallel  \cdot  {\parallel }_{\varepsilon }$ ,使得

$$
\parallel \mathrm{A}{\parallel }_{\varepsilon } \leq  \mathbf{\rho }\left( \mathrm{A}\right)  + \varepsilon  = \frac{1}{2}\left\lbrack  {\mathrm{r} + \mathbf{\rho }\left( \mathrm{A}\right) }\right\rbrack   < \mathrm{r}
$$

$$
{\begin{Vmatrix}{\mathrm{c}}_{\mathrm{k}}{\mathrm{A}}^{\mathrm{k}}\end{Vmatrix}}_{\varepsilon } \leq  \left| {\mathrm{c}}_{\mathrm{k}}\right| \parallel \mathrm{A}{\parallel }_{\varepsilon }^{\mathrm{k}} \leq  \left| {\mathrm{c}}_{\mathrm{k}}\right| {\left\lbrack  \mathbf{\rho }\left( \mathrm{A}\right)  + \varepsilon \right\rbrack  }^{\mathrm{k}}
$$

当 $\left| z\right|  < r$ 时, $\sum \left| {c}_{k}\right| {\left| z\right| }^{k}$ 收敛，于是

$\sum \left| {c}_{k}\right| {\left\lbrack  \rho \left( A\right)  + \varepsilon \right\rbrack  }^{k}$ 收敛 $\Rightarrow  \sum {\begin{Vmatrix}{c}_{k}{A}^{k}\end{Vmatrix}}_{\varepsilon }$ 收敛 $\Rightarrow  \sum {c}_{k}{A}^{k}$ 收敛

(2)设 $A$ 的特征值 $\lambda$ 满足 $\left| \lambda \right|  = \rho \left( A\right) , x$ 为 $\lambda$ 相应的特征向量

$$
\mathop{\sum }\limits_{{k = 0}}^{n}{c}_{k}\left( {{A}^{k}x}\right)  = \mathop{\sum }\limits_{{k = 0}}^{n}{c}_{k}\left( {{\lambda }^{k}x}\right)  = \left( {\mathop{\sum }\limits_{{k = 0}}^{n}{c}_{k}{\lambda }^{k}}\right) x
$$

由于 $\rho \left( A\right)  > r$ ,那么 $\left( {\mathop{\sum }\limits_{{k = 0}}^{n}{c}_{k}{\lambda }^{k}}\right) x$ 发散(注意 $x$ 为非零向量)

从而 $\mathop{\sum }\limits_{{k = 0}}^{n}{c}_{k}\left( {{A}^{k}x}\right)$ 发散,这样 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{A}^{k}$ 发散

定义: 设一元函数 $\mathrm{f}\left( \mathrm{z}\right)$ 能展开为 $\mathrm{z}$ 的幂级数

$$
f\left( z\right)  = \mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{z}^{k}\;\left( {\left| z\right|  < r, r > 0}\right)
$$

其中 $\mathrm{r} > 0$ 表示该幂级数的收敛半径。当 $\mathrm{n}$ 阶矩阵 $\mathrm{A}$ 的谱半径 $\mathbf{\rho }\left( \mathrm{A}\right)  < \mathrm{r}$ 时, 把收敛的矩阵幂级数 $\mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{A}^{k}$ 的和记为 $f\left( A\right)$ ,即

$$
f\left( A\right)  = \mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{A}^{k}
$$

性质 (代入规则): 若 $f\left( z\right)  = g\left( z\right)$ ,则 $f\left( A\right)  = g\left( A\right)$ .

例1:

$$
{\mathrm{e}}^{\mathrm{z}} = 1 + \frac{1}{1!}\mathrm{z} + \cdots  + \frac{1}{\mathrm{k}!}{\mathrm{z}}^{\mathrm{k}} + \cdots \;\left( {\mathrm{r} =  + \infty }\right)
$$

$$
{\mathrm{e}}^{\mathrm{A}} = \mathrm{I} + \frac{1}{1!}\mathrm{A} + \cdots  + \frac{1}{\mathrm{k}!}{\mathrm{A}}^{\mathrm{k}} + \cdots \left( {\forall {\mathrm{A}}_{\mathrm{n} \times  \mathrm{n}}}\right)
$$

$$
\sin z = z - \frac{1}{3!}{z}^{3} + \cdots  + {\left( -1\right) }^{k}\frac{1}{\left( {{2k} + 1}\right) !}{z}^{\left( 2k + 1\right) } + \cdots \;\left( {r =  + \infty }\right)

## 三、矩阵函数的一般定义

展开式 $f\left( z\right)  = \sum {c}_{k}{z}^{k},\;\left( {\left| z\right|  < r, r > 0}\right)$ ，要求

(1) ${f}^{\left( k\right) }\left( 0\right)$ 存在 $\;\left( {k = 0,1,2,\cdots }\right)$

(2) $\mathop{\lim }\limits_{{k \rightarrow  \infty }}\frac{{f}^{\left( k + 1\right) }\left( \xi \right) }{\left( {k + 1}\right) !}{\xi }^{k + 1} = 0\;\left( {\left| z\right|  < r}\right)$

对于一元函数 $f\left( z\right)  = \frac{1}{z}$ 等,还不能定义矩阵函数。

基于矩阵函数值的Jordan标准形算法, 拓宽定义

矩阵函数的一般定义

设 ${P}^{-1}{AP} = J = \operatorname{diag}\left( {{J}_{1},\cdots ,{J}_{s}}\right) ,{J}_{i} = {\lambda }_{i}I + {I}^{\left( 1\right) }$

如果 $f\left( z\right)$ 在 ${\lambda }_{i}$ 处有 ${m}_{i} - 1$ 阶导数,令

$$
f\left( {J}_{i}\right)  = \mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{J}_{i}^{k} = f\left( {\lambda }_{i}\right) I + \frac{{f}^{\prime }\left( {\lambda }_{i}\right) }{1!}{I}^{\left( 1\right) } + \cdots  + \frac{{f}^{\left( {m}_{i} - 1\right) }\left( {\lambda }_{i}\right) }{\left( {{m}_{i} - 1}\right) !}{I}^{\left( {m}_{i} - 1\right) }
$$

$$
f\left( A\right)  = \mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{A}^{k} = P \bullet  \mathop{\sum }\limits_{{k = 0}}^{\infty }{c}_{k}{J}^{k} \bullet  {P}^{-1} = P \bullet  \operatorname{diag}\left( {f\left( {J}_{1}\right) ,\cdots , f\left( {J}_{s}\right) }\right)  \bullet  {P}^{-1}
$$

称 $f\left( A\right)$ 为对应于 $f\left( z\right)$ 的矩阵函数

[注] ① 拓宽定义不要求 $\mathrm{f}\left( \mathrm{z}\right)$ 能展为 “ $\mathrm{z}$ ”的幂级数,但要求在 $\mathrm{A}$ 的特征值 ${\lambda }_{\mathrm{i}}$ (重数为 ${\mathrm{m}}_{\mathrm{i}}$ ) 处有 ${\mathrm{m}}_{\mathrm{i}} - 1$ 阶导数,后者较前者弱!

② 当能够展为 “ $\mathrm{z}$ ”的幂级数时,矩阵函数的拓宽定义与级数原始定义是一致的.
